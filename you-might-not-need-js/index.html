You Might Not Need JS

Websites are bloated, JavaScript bundle sizes are huge, and Server Side Rendered apps are plain unusable until they're hydrated (why bother).

We have not realised yet that our users may not have JavaScript. Yet we use it to build simple things like dropdowns, navigation or to submit forms that could be built with CSS or without JS at all.
